# 3D Interactive Quantum Matrix // Task 2

An advanced, highly responsive 3D interactive landing page built for the Techfest Web Development domain. This project implements a high-performance 3D particle engine rendered on an HTML5 canvas, completely driven by math and native browser APIs without relying on heavy external frameworks like Three.js.

## ⚡ Core Engineering Features
* **Native 3D Particle Engine:** Utilizes advanced trigonometric calculations ($\sin$, $\cos$, and matrix projections) to render an active, rotating 3D particle singularity sphere in real-time.
* **3D Scroll Morphing & Transitions:** As the user scrolls down the page, the gravitational properties of the matrix update dynamically, morphing the sphere into a flattened compression ring array, and finally exploding outwards into a widespread data burst.
* **Proximity Vector Linking:** Implements a Pythagorean proximity algorithm that calculates the distance between floating nodes in real-time, dynamically drawing glowing neural connection ribbons between neighboring data clusters.
* **Cursor Gravity Tracking:** Features fluid, interpolated mouse tracking that allows users to warp and rotate the core orientation plane by moving their cursor across the viewport.
* **Live System Monitoring Overlay:** Includes an integrated operational readout dashboard that computes and shifts data integrity metrics smoothly on a continuous oscillation tree.

## 🛠️ Built With
* **HTML5:** Semantic workspace containers and canvas layering.
* **CSS3:** Glassmorphic card overlays (`backdrop-filter`), layout alignment, and neon text configurations.
* **Vanilla JavaScript (ES6+):** 3D perspective calculation loops, proximity detection algorithms, and event interpolators.
